# Spotify

This repository contains the implementation for the LI3 project of the year 2024.

Phase 1 limits:
- Time: 5 minutes
- Memory: 2 GB

Phase 2 limits:
- Small dataset:
    - Time: 5 minutes
    - Memory: 2 GB
- Large dataset:
    - Time: 11 minutes
    - Memory: 7GB

You can use the following command in your terminal to impuse the memory limits:

```bash
# limit the memory usage by 2 GB
ulimit -m 2048000
```
